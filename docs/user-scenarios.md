# Пользовательские сценарии (ЛР10)

## Сценарий 1: Аналитик — полный цикл

1. Вход: `analyst@demo.local` / `analyst123`
2. Подключение: демо PostgreSQL (schema `demo`) или своё через форму `/connect`
3. Вопрос: «Покажи продажи по категориям за последний квартал»
4. Система (mock LLM) возвращает SQL → пользователь при необходимости правит
5. Выполнение → таблица + столбчатый график
6. Сохранение дашборда

## Сценарий 2: Просмотр (viewer)

1. Вход: `viewer@demo.local` / `viewer123`
2. Открытие `/dashboards` → выбор готового дашборда
3. Только просмотр таблицы и графика (без SQL/сохранения)

## Таблицы демо

- `demo.categories` (id, name)
- `demo.sales` (id, category_id, amount, sold_at)
